1) Setup - top

To ensure everything works correctly, please use <!DOCTYPE html>

Include the following code into HEAD section of the page

Include CSS file

<link rel="stylesheet" href="css/openpanel.css" type="text/css" />

Add Viewport Meta:

<meta name="viewport" content="width=device-width, user-scalable=no" />

Include Javascript files

 <!-- jQuery Lib -->

<script type="text/javascript" src="js/jquery/jquery.min.182.js"></script>


<!-- mousewheel plugin -->

<script type="text/javascript" src="js/jscrollpane/jquery.mousewheel.min.js"></script>


<!-- jScrollPane script -->

<script type="text/javascript" src="js/jscrollpane/jquery.jscrollpane.min.js"></script>


<!-- OpenPanel -->

<script type="text/javascript" src="js/openpanel.dev.js"></script>

Setup done.

2) Create Panel - top

With Open Panel plugin you can use two methods to show panels ( Inline Content and AJAX content ).

If you use Inline Content Method you will need to create panel content in viewing page and place it in Open Panel Station block.

If you use AJAX Content Method you will need to create new files to contain panel content and place them in "/content/openpanel" folder.

Now we will start to create a sample panel.

First, we create a block and give it an id and and a class="op-panel": 

<div id="panelname" class="op-panel">
  <div class='op-panelform'>
    <!-- Panel Content -->
  </div>
</div>

Then we need to add a close button to close the panel.
Note: the close button must have "data-close value" (this is the current panel id.) and op-bt-close class. 

<div id="panelname" class="op-panel">

  <!-- Control -->
  <div class="op-panelctrl">
    <!-- Close Button -->
    <div data-close='panelname' class="op-panelbt op-bt-close">
      <img src="images/arrow-left-48.png" alt="close" />
    </div>
    <!-- End Close button -->           
    <div class="clearspace"></div>
  </div>
  <!-- End Control -->

  <div class="op-panelform">
  <!-- Panel Content Here -->
  </div>

</div>

Final, You can add any content into the panel. You can see the demo file in the downloaded package for more detail.

3) Call The Plugin - top

To open the panel you just add data-panelid="panelname" and a class you use to open panel ( Example: op-elem.). Additional, you can add data-pos, data-pbg and data-ptext if you like. See Parameters Section for more detail.

Example:

<div data-panelid="panelname" data-pos="none" data-pbg='solid-darkgreen' data-ptext='light-text' 
class="op-elem">
  Open Panel
</div>

Then add javascript to call the plugin into <head> tag: 

<script type="text/javascript">
$(window).load(function(){
    $('.op-elem').openpanel();
});
</script>

4) Parameters - top

Javascript Parameters:

Parameters Properties Descriptions

HTML Parameters

Parameters Properties Descriptions
data-panelid string This param is used to determine which panel will be opened.
data-pos string This param is used to set display effect.
Value: top, left, right, bottom, none.
data-pbg string This param is used to set background for panel.
Value: See Background list in CSS.
data-ptext string This param is used to set text color for panel.
Value: light-text, dark-text.
data-close string This param is used to close panel.
Value: The current panel id.
